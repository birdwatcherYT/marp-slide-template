# marpスライドテンプレート

## スライドの生成方法
```
$ make help
使い方:
  make pdf                           slide.md をPDFに変換
  make html                          slide.md をHTMLに変換
  make docs                          slide.md をHTMLに変換しdocs/index.htmlに反映
  make preview                       slide.md をプレビュー
  make all                           slide.md をPDFとHTMLに変換
  make <command> TARGET=<ファイル名> 任意のファイルを変換
```
